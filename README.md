# Redshifter

Redshifter is a synthesizer developed using the JUCE framework.

It contains sine and sawtooth oscillators with envelopes and 8 voice polyphony.

It also has reverb, chorus, phaser, gain, and a filter with an LFO to modulate the cutoff frequency.
